## 修改：Footer 动态读取 git 信息

**commit**: `5f1afbf`

**改动内容**：将 3 个 footer 文件中的硬编码 "2026-02-06 | 106" 改为从 git 动态读取。

**实现方式**：
- `next.config.mjs` 使用 `child_process.execSync` 在 build 时执行：
  - `git rev-list --count HEAD` → 提交次数
  - `git log -1 --format="%ci"` → 最新提交日期（取 YYYY-MM-DD 部分）
- 通过 Next.js `env` 注入 `NEXT_PUBLIC_GIT_COMMIT_COUNT` 和 `NEXT_PUBLIC_GIT_LAST_DATE`
- 3 个文件（`page.tsx`, `essays/[essayId]/page.tsx`, `Sidebar.tsx`）使用 `process.env.NEXT_PUBLIC_*` 读取

**验证**：本地 `npm run build` 成功，HTML 中正确输出 `2026-02-06` 和 `106`。
