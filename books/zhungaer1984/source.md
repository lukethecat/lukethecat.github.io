# 准噶尔诗草

## 作者小传

李瑜，当代著名西部诗人，原籍安徽巢县，生于重庆，长在武汉，浪迹了大半个中国，但一生中大部光阴却是在天山北麓的风雪和浓荫中度过的。爱与死的永恒主题和他结下不解之缘。他的诗及其创作生涯曾由中国国际广播电台以十多种语言对外播发介绍，并翻译成多种国内少数民族文字。著有诗集《准噶尔诗草》《啊，伊犁河水漂白了我的军衣》《战争与城》《为了爱情，巴格达不嫌远》和散文集《小巷系列》等。

## 内容简介

《准噶尔诗草》是李瑜早期诗歌代表作，收录了诗人在新疆垦区创作的抒情诗篇。全书分为四卷：飘动的篝火、将军与城、一位诗人在古尔班通古特、为了爱情巴格达不嫌远。诗集真实记录了开拓者在准噶尔盆地、古尔班通古特沙漠的创业历程，展现了边疆建设者的理想、劳动与爱情。出版社：新疆人民出版社，1984年9月第1版。

## 开拓精神万岁（序言）

**公刘**

新疆人民出版社一下子给我邮来了十部诗稿，嘱我为这个行将问世的丛书作一篇序。我放下手头的一切工作，花了整整半个月时间，逐字逐行地通读了一遍，能从歌声中结识这么一个庞大的新疆诗人群——既有五十年代、六十年代、七十年代开始写作的中年诗人和青年诗人，也有兄弟民族诗人，真是三生有幸；我没法子不激动，感触颇多，反而不知道该从何说起了。

去年九月间，我去过一趟新疆；但由于所在单位一再电报催促回家参加机构改革，我被迫中断了原定的日程，把终点暂且放在伊宁，调转头便回了乌鲁木齐，来去匆匆，见面有限；然而，纵使这样，我也自信领略了一点新疆特有的诗意。

新疆特有的诗意是什么？闭目凝思，窃以为，“开拓”二字，也许可以概括。

记得旅途之上，每当我问起下一站在哪儿歇脚时，得到的答复总是，同样的一句：再走×百里。大矣哉！新疆！这种雄伟的气魄和寥廓的境界，给了我十分强烈的印象。

应该说，收在这套丛书中的大部分诗作，一般都能反映这一特色。无论冰峰、火洲、草原、湖泊、沙漠、戈壁、毡房、畜栏、开垦、狩猎、葡萄园、坎儿井、胡杨、塔松、清真寺、巴扎、叼羊、姑娘追、热瓦甫、冬不拉……端的是这一方的水土和这一方的景观，这一方的人物和这一方的心态。

没有，或者缺乏亲身体验的人是肯定写不出来的。这又印证了一句话：生活是创作的唯一源泉。可能会有人摇头：老一套！

不错，被实践反复检验过的真理，往往像是老生常谈。然而，如果人们真的把它当作了老生常谈对待，那就未免太轻率了；可以断言，受害者不会是别人，只能是自己。

生活是没有穷尽的，任何人都无权夸口：我的生活已经足够了。永远没有足够的时候。何况，爆发在生活中的诗的灵感，从来都是一个害羞的、胆怯的天使，她在你的视野中，绝不会原封不动地出现第二次。诗人的任务是，及时捕获这稍纵即逝的精灵，用文字加以固定，或者说，用文字加以复印。

一切全靠生活底蕴的厚实。我想，在这套丛书的作者中，凡是取得了成功的同志，都是生活的有心人。

尤其值得珍贵的是，新疆的生活既是开拓者的生活又是正在不断开拓着的生活。在人与自然的关系中，开拓意味着进取和奋斗，意味着无所畏惧，意味着寄希望于未来，因此，它需要理想、勇气与毅力。新疆的诗人理所当然地应该是开拓者的气质。在人与艺术的关系中，开拓意味着探索和创造，意味着革新，意味着绝不认为每件事情都有了现成的答案，因此，开拓者诗人所坚持的现实主义，就不应该仅仅是一种创作方法，而应该是一个开放的艺术思想体系，它要显示包罗万象、消而化之的气度，它要恪守以我为主、为我所用的原则。

纵观新疆近年的诗坛，确实令人欢欣鼓舞。一些有代表性的诗人写下了一些有代表性的篇章，它们有明明白白的时代感，扎扎实实的历史感，它们有胸襟、有力度、有严峻豪迈的美，有粗犷与细腻的适当揉合，最重要的是，它们有明天。

展望未来，我对新疆的新诗运动的前景是乐观的。乐观的最大根据是：新疆有人才，因为开拓出人才，只要不丢掉开拓精神，新疆的诗歌就一定会有更灿烂更广阔的前途。

是为序。

**1984年1月11日于合肥**

## 卷之一：飘动的篝火（29首）

### 开镰歌

```
割啊，割啊，割啊，
割下一片红霞，割下万层麦浪；
我们终于把这一天盼来了，
金黄的穗子就碰在笑脸上。

康拜因的银镰早就磨好了，
丰收的歌儿早就在心里欢唱；
你摸摸我滚烫的胸口吧，
心跳得就像急骤的鼓点一样。

在这片土地上开拓，
在这片土地上生息；
谁说是天山水把禾苗滋长？
这麦粒是以我们的心血灌浆。

割啊，割啊，割啊，
割下一片红霞，割下万层麦浪；
不要说我们只经历了一个寒暑，
在很久前就把幸福的种子播上。
```

**1972年5月**

### 爆繁开

> 在古尔班通古特瀚海扬帆

```
康拜因在朦胧的晨曦中启航了，
迎着闪动的金色波涛；
“三八”机车组好气派，
在古尔班通古特瀚海扬帆。

掠过万顷汹涌的金浪花，
赢得阵阵喝彩、声声惊叹；
“我们姑娘的银镰一晃，
‘解放牌’又拉走一座金山。”

粮食丰收了，歌儿也丰收了，
粮食流满粮仓，歌儿流满空间；
明年的粮食还将填满准噶尔盆地，
歌声将把河流、天山连同梦境都一起填满。

乘着和煦的夏风，
乘着芬芳的夏风，
康拜因手在古尔班通古特瀚海穿梭，
不倦地编织锦绣般的塞外江南。
```

**1972年5月**

### 春风早已渡过玉门关了

```
春风早已渡过玉门关了，
巍巍天山依然戴着晶莹的雪冠一顶；
给栽好的树苗再培上一锹沃土吧，
以赤子的绵绵深情。

塞外的白桦，华北的青松，
江南的云杉，中原的泡桐，
都来茫茫的大漠扎根，
都来遥远的大漠扎根。

这儿的阳光与故乡一样灿烂，
这儿的天地比故乡更加辽阔；
根儿扎得紧，
枝叶更茂盛。

不只是栽下几行青青的树苗，
分明筑起陡峭的绿色长城；
那时，傲风沙，击雷霆，
支撑苍穹，高耸入云。

不要看五湖四海、各路乡音，
热爱边疆都有一颗赤诚的心；
祖国啊，请来远方选栋梁，
一代新林正在时代的风雪中长成。
```

**1972年5月**

### 挺进大西北

```
你收到过许多的远方来信，
可是从来没有像这样遥远；
这封信将横穿辽阔的大地，
又飞回曾哺育过我的江南。

沿着军用地图上的红色箭头，
挺进大西北，
豪迈而且庄严；
这是一次伟大的历史性进军，
绿色革命的狂飙将席卷边塞。

还记得当年曾谈过志愿，
谈论过神秘的大漠，
谈论过遥远的天山；
这一天思盼了如此的长久，
渴望战斗的理想即将在这儿实现。

夜空升腾召唤出击的信号，
——大漠的风送来纷扬的雪片；
啊，又一个西北战场大军集结的不眠之夜，
我也有那无畏的魂胆……
```

**1973年5月**

### 在地窝子里写下了这首小诗——沙包纪事

> 某部驻地有两个沙包，人们亲昵地称呼这个地方为“沙包市”。  
> ——摘自手记

```
要经过几十年，也许是半个世纪，
大漠边缘将要出现一座崭新的城市；
那时，在新出版的袖珍地图上面，
将正式写上我们曾深情呼唤过的名字。

“青年大街”将响彻钢铁的交响，
“绿洲广场”将奔腾彩色的旋律；
再也看不到脊背上印着梅花的小鹿了，
也看不到火一样的红柳、翡翠般的骆驼刺。

我们留下一片绿洲和一张蓝图，
还有垦荒志愿队的那面褪色的红旗；
这座新城就像远航中的一个港湾，
将从这儿去征服瀚海——茫茫戈壁。

那时，我们早就不在这儿居住了，
早就进驻了新的“沙包市”；
请记住二十世纪这个不平凡的夜晚，
一个拓荒者在地窝子里写下了这首小诗。
```

**1973年5月**

### 趁着天山月——致阿不都

> 1951年初，在开发石河子垦区的日日夜夜，垦荒农具坎土曼主要依靠维吾尔青年铁匠阿不都锻打修理。

```
我好像又看到你屹立在打铁炉边，
绣花帽上还映着西北战场的硝烟；
你掏出全身的力量献给垦区，
趁着天山月，
再打一把坎土曼。

请在锋刃上多加上一些好钢吧，
披荆斩棘的战士将要重新安排河山；
在这儿，我们曾一起流过鲜血，
还将在一起辛勤耕耘，
以汗水浇灌。

在万古荒原撒下簇簇灿烂的钢花，
迎着古尔班通古特金黑色的黎明飞溅；
这不仅使人想起缤纷的胜利花雨，
还使人想到挥动的军刀，
还使人想到曳光的子弹。

你拉着风箱——鼓动胸中的风雷，
炉火中燃烧着愤怒的复仇的火焰；
侧耳倾听，
巴里坤追剿残敌的枪声，
铁锤还要把旧世界最后一个巴依打翻。
```

**1973年5月**

### 啊，红柳

```
啊，红柳，
你的叶片从容漫卷西风，
你的枝干愤怒击碎乌云，
像火红的旗帜飘扬在茫茫大漠，
茫茫大漠呼啸着将你高擎；
灿烂的阳光照抚着你，
晶莹的雨露滋润着你，
放射夺目的光芒，
闪烁胜利的豪情。

啊，红柳，
即使再过若干世纪你也坚信：
有一天，能锁住黄沙万顷。
为此默默献出了一切，
哪顾躯干又增添了几圈年轮。
你从摇曳着萌芽的叶片开始，
就高唱进行曲，豪迈而且热情；
将生命的种子撒向古尔班通古特腹地，
迎着漫天风沙呐喊着前进。
```

**1973年5月**

### 闪耀着月光

```
闪耀着月光，
闪耀着星光；
雪浪簇拥，唱着抒情的夜歌，
拓荒者以汗水粘合了每一寸渠墙。

夜也不平静，生命都在滋长，
欢腾的雪浪是准噶尔跳跃的血浆；
苞谷拔节的声音这么清脆，
惊走了调皮的小鹿、胆小的黄羊。

捎来塔松的一片涛声，
捎来雪莲的一缕幽香；
为了一个信念而不息地奔流，
在漫长的征途吸收了多少热量。

汇聚着月光，
汇聚着星光；
奔腾的雪浪——祖国母亲香醇的乳泉，
日夜哺育着辛勤开发的绿色的原野。
```

**1973年7月**

### 燃起一堆飘动的篝火——致梅花鹿

```
在这儿，
我们燃起一堆飘动的篝火，
大漠的新纪元就从这闪光的时刻开始；
喧闹的谈吐也有划时代的意义，
因为这儿自古以来就人迹罕至。

梅花鹿在月下睁开惺忪的睡眼，
开发第一块绿洲时我们就似曾相识；
不要说打扰了你在红柳丛中的好梦，
我们还要带来奔腾的雪浪，
我们还要带来淅沥的雨滴。

大漠深处的红柳也像火一样燃烧，
还像火一样热情，
会慷慨让你在她的脚下暂住一时；
你还会得到一顶宽阔的青纱帐，
——一片翡翠般的骆驼刺。

还是乘着朦胧的月光离开这儿，
不要再向我们张望了；
再也不要迟疑；
(也许是留恋暖热了的一片黄沙，
已经在这儿居住了好多个世纪。)

去吧，去吧，
我们还会再见，
我们开拓，我们前进，
——这是一首没有尾声的史诗；
飘动的篝火还要闪耀在古尔班通古特腹地。
```

**1973年7月**

```
已经告别了锦绣般的稻田，
已经告别了毗连着的新村，
——并不是江南故乡，
几年前我们才来到那儿。

依然还是褐黄的峰峦，
依然还是褐黄的浮云；
沉睡着的大漠在召唤，
拓荒者的人流像瀚海潮般汹涌奔腾。

未来的团部在哪儿呢？

部队就该在这个沙包下扎营。

连首长兴奋地向四周指指划划，
一片绿洲便将在这儿诞生；
今夜，也许会在沙包下做个绿色的梦，
这将是明日向大漠开战的蓝本。
```

```
曾走过石板桥，
曾走过独木桥；
在人生的路上曾走过多少桥，
今夜又走红柳桥。

古老的荒原已经破晓，
```

## 跋：从大漠升起袅袅的笛声——读《准噶尔诗草》

**阿红**

### A

李瑜邀我为他即将出版的处女集《准噶尔诗草》写篇读后。

我心怵。

哪能不怵呢？

车尔尼雪夫斯基说：“关于诗人的性格、生平和他所接触的人们，我们知道的愈多，就愈能在他的作品中看出活人的肖像。”

我后悔去年九月在“绿风”诗会与李瑜相识时，没做个有心人多和他谈谈。那是一个了解他的生平的机会哪！

而今天遥地远，云飞鹰旋，迢迢万里，问答艰难。

仅仅只能从他早期发表诗作署名前的仿宋字知他曾是“知识青年”。

仅仅只能从一篇评论边塞新诗的文章里，知他在“史无前例”的日子里“曾度过一段悲怆的流浪生活”；知他从绿洲新城被放逐到古尔班通古特沙漠边缘的一个军垦农场，于1972年唱着《开镰歌》走上诗坛的。

仅仅知他最近结束了二十余年的教学生涯，调往乌鲁木齐市一家报社工作。他负责的那副刊名家如星，编得出色。

再呢，就是这部《准噶尔诗草》。

种种信息互相碰撞、融会，我对李瑜得出了如下的印象：

他是地地道道垦区的建设者和绿洲的开拓者。在准噶尔，在古尔班通古特，他在那儿辛勤地劳动着，他在那儿抛洒过汗滴，说不定在那儿也抛洒过泪滴呢！

他有理想，有信念，有落到大海也不会沉没的精神。

他劳动着，生活着，写着诗。开拓绿洲的战场上，他是随军记者，是拿坎土曼的诗人。

他有不少的梦。他也会做梦。他在物质的荒原上，负着精神上的重压，为实现自己的梦，他以“为了爱情，巴格达不嫌远”的精神，不屈不挠地拼搏着、跋涉着。应该说，他的梦在一一实现。

或许说：李瑜生活之路也够崎岖坎坷的。可也是，请原谅我说句不讨人喜欢的话，这崎岖坎坷的生活之路倒也成全了李瑜。试想古今诗人，有几位是糖水里泡大的？相反，大多有过不平常的经历。王安石说：“岁老根弥壮，阳骄叶更明。”拜伦也说：“逆境是达到真理的一条通路。”别林斯基也说：“不幸是一所最好的大学。”李瑜从南国故乡到新疆开拓绿洲，投入了将永远被史籍称道的开发新疆的战斗。当然，生活是艰苦的，不正是这艰苦的生活，使他深切地熟悉了、认识了开拓者吗？使他锤炼了意志、心灵吗？使他献出这本《准噶尔诗草》吗？使他成为一颗诗的新星吗？

这本《准噶尔诗草》，

就是他人生经历的画卷，

就是他精神境界的荧光屏，

就是他从大漠升起的袅袅的笛声！

### B

想当年俄罗斯诗人涅克拉索夫也曾写过些被批评家别林斯基认为是人云亦云的诗稿。这时的涅克拉索夫是在诗创作的童年阶段。

我漫读李瑜的诗，发现他也有创作的童年阶段。

《开镰歌》是他问世的处女作。1972年发表的，免不了还有那个时期的生活印记。这次收集放在首篇，可能是出于纪念的心情。

```
割啊，割啊，割啊，
割下一片红霞，割下万层麦浪；
我们终于把这一天盼来了，
金黄的穗子就碰在笑脸上。

康拜因的银镰早就磨好了，
丰收的歌儿早就在心里歌唱；
你摸摸我滚烫的胸口吧，
心跳得就像急骤的鼓点一样。

在这片土地上开拓，
在这片土地上生息；
谁说是天山水把禾苗滋长？
这麦粒是以我们的心血灌浆。

割啊，割啊，割啊，
割下一片红霞，割下万层麦浪；
不要说我们只经历了一个寒暑，
在很久前就把幸福的种子播上。
```

这首诗的构思、语言都一般，思想开掘也较浮浅，是的，我看到的这个集子初选稿里确实有一些这样的诗。

我指出这个问题，是因为我想到任何一个文学刊物接到的诗稿里有不少这样的诗，情绪健康，结构完整，可就是少了一点作为诗歌艺术必不可少的东西。

我是认为抒情诗，终究抒的是诗人之我的情思，终究是以诗人之我的方式抒诗人之我的情思。诗，是诗人这个人写的。他在某种境遇里对某种事物有所见，有所感，有所思，就用诗来坦露自己的心灵，传达自己的心灵信息。勿论他是自觉或是不自觉，是想使别人也能见诗人所见，感诗人所感，思诗人所思。因此，抒情诗这种艺术强烈要求显示诗人的个性。

戴复古说：“须教自我胸中出，切忌随人脚后行。”

何绍基说：“诗是自家作的，便要说自家的话，凡可以彼此公共通融的话头，都与自己无涉。”

高尔基说：“谁想当作家，谁就应当在自己身上找到自己。”

黑格尔说：“人一旦从事于表达他自己，诗就开始出现了。”

当然，诗人要心灵美，志趣高，胸襟宽，理想远。他是大海里的一滴水，和大海同温凉起伏，从他这一滴水里能发现大海。

李瑜是有能耐的。他快步跨过了创作的童年阶段。

无论抒情主人公是“我”，还是他讴歌的开拓者和建设者——解放新疆而血染戈壁的战士、接过父辈悲怆的遗言的维吾尔族垦荒队员、被流放的诗人、多情的拖拉机手……无论是展现开拓者的理想——憧憬的绿洲、对新城的呼唤、在大漠里企盼灿烂文明的复兴、将蓝宝石缀在祖国母亲的裙衫……劳动——植下一条翡翠般的林带、为大漠点燃一个新世纪的篝火、在茫茫的古尔班通古特拉开艰难的第一犁、于经纬仪上瞄呀瞄呀的勘测者……爱情——初恋、别离、思念、焦灼、痛苦、喜悦……无论那孕生诗情的境遇是大漠、是原野、是新城、是黎明、是星夜……

我们都能发现李瑜有着自己的眼睛、耳朵，有着自己的头脑、心灵。

他写着自己的官能感觉。像他写沙漠林带：

```
拥簇着层层叠叠的浪花，
在夕阳映照下奔腾着、喧哗着；
他的涛声就是大地轻轻的呼吸，
那么妩媚，
那么柔和……
```
——《流淌着一条翡翠的小河》

显然，这样的感觉、印象、联想、描绘，是他的。

他写着自己的情思，自己的心灵搏动。像他表达开拓者不断向沙漠挺进的精神，便是由燃着了篝火惊动了一只梅花鹿，而向这只梅花鹿情不自禁地嘱咐体现的：

```
不要说打扰了你在红柳丛中的好梦，
我们还要带来奔腾的雪浪，
我们还要带来淅沥的雨滴。

大漠深处的红柳也像火一样燃烧，
还像火一样热情，
会慷慨让你在她的脚下暂住一时；
你还会得到一顶宽阔的青纱帐，
——一片翡翠般的骆驼刺。

还是乘着朦胧的月光离开这儿，
不要再向我们张望了，
再也不要迟疑；
(也许是留恋暖热了的一片黄沙，
已经在这儿居住了好多个世纪。)

去吧，去吧，
我们还会再见，
我们开拓，我们前进，
——这是一首没有尾声的史诗，
飘动的篝火还要闪耀在古尔班通古特腹地。
```
——《燃起一堆飘动的篝火——致梅花鹿》

显然，这是他这个开拓者、创业者的情思。他的伙伴见了这只梅花鹿可能是嬉戏追逐，而他浮想联翩，情不自已。

我不知道李瑜是如何能那么快跨越创作的童年阶段的。

也许他是从中外名家诗篇领悟了诗的诀窍；  
也许他是从中外名家诗论体味了诗的秘密；  
也许他是从自己创作实践不断理解了诗的方式；  
也许是这些的综合。

这只有李瑜能说清楚，甚至他也不能说清楚。

横竖跨越了这个阶段：

他才告别了一般化，有了打动人心的感情，他才有可能形成自己的艺术风格。

李瑜是智慧的。智慧的头脑会更倾听诗的话、理论的话、生活的话和自然的话。

### C

是的，李瑜的诗有了自家的面目。

读着他的那些诗，仿佛听着一位有着流泉一样性格的人，轻声慢语地同我娓娓叙谈。他说话有自己的腔调；笑，也有自己的情状。

我认为人的自然个性和社会个性制约着人的艺术个性。诗不是人写的吗？人写着诗，他的经历、他的性格、他的人品、他的气质、他的风度，不管他有意还是无意，就遗落在诗里。像鸟在空中飞，影子在地上跟着飞一样。杜甫别于李白，李白别于王昌龄，王昌龄别于贺知章。艾青别于臧克家，臧克家别于贺敬之，贺敬之别于郭小川。一个抒情诗人越是能够在诗里写我之情思，用我之方式写我之情思，那么，他在诗里留下的影子就越是深刻、鲜明。

不过，我也并不认为一个诗人的自然个性、社会个性就全等于这个人的艺术个性。我观察过一些诗人。有的是火性子，点火就着，可他的诗却那么温柔、委婉，虽然有时候，也有点烈性。接触多了，我了解到这个人在家庭在和朋友交往中却是个感情细腻的人。有的人从诗里看电闪雷鸣，风掣云驰，而他给人的印象却像只温驯的猫咪，轻声细语，妙呜妙呜的。但他办事、处理问题，诚是柔中有刚，泼辣得很。诗，显示着人最内在的个性。同时，制约着一个诗人的艺术个性的，还有他的审美趣味、艺术追求。像我说的这位像猫咪的诗人，他就喜欢大江东去般的诗，无形中自己的诗作就受到苏东坡、辛弃疾、惠特曼的熏陶。这也是顺理成章的事。

我和李瑜于石河子、乌鲁木齐匆匆地聚散，对他的个性了解甚少，不敢妄言。从他的诗看，从他那些卓具特色的诗看，我的审美总印象是：自然、俊秀、婉曲。

**自然**，我理解的自然，一是情思生得自然。男女约会，一个到了，一个迟迟不到，那早到的就引颈了望，就焦灼，就作种种设想。这种情思就生得自然。杜勃罗留勃夫说的：“在抒情作品中表现了某种现象在诗人身上唤起的自然的感情”，也就是这个意思。

一是情思写得自然。按着情思的形态、节奏、路儿去写，不装饰，不捏弯，云舒于天，水行于地。像在《戴着高高花冠的小鸟》里，作者凝望着飞过正在开垦的荒原的这鸟儿絮絮说着：“不要嫌这儿还没有一片绿色的林海，没有红艳艳的花，没有绿茵茵的草，也没有燃烧着青春之火的抒情交响乐章，——那曾永远伴着你的飒飒风涛。留下来吧，就在这儿筑下一个小巢吧，这儿就会成为茫茫瀚海中的一个绿岛；虽然这儿只有金字塔般的褐色浮雕，可是已摇曳象征我的定居的翡翠的树苗……”深情的话语就像汩汩流着的清泉一样。自然的生命是真实，是抒情主人公在特定境遇中情绪、感情、思想的真实。爱情、友情、乡情得能自然，那不难；写开拓之情、创业之情，而臻于自然，那是要难得多的，因为得真有那种情愫。

**俊秀**：我是说他的那些好诗虽然并未特别梳妆打扮，但是眉清目秀，有神采，有风致，有几分妩媚。像他在《闪耀着月光》里，写月下流过绿洲的水渠：“夜也不平静，生命都在滋长，欢腾的雪浪是准噶尔跳跃的血浆；苞谷拔节的声音这么清脆，惊走了调皮的小鹿、胆小的黄羊。捎来塔松的一片涛声，捎来雪莲的一缕幽香；为了一个信念不息地奔流，在漫长的征途吸收了多少热量。”像他在《啊，我心中有一轮永远不落的月亮》的诗行，“红柳还在摇曳，夜莺还在歌吟，温柔的夜曲般的絮语还在萦绕。她早就离开这儿了，如白云般轻盈，悄悄离开了这绛紫的纱帐。”都给读者一种俊秀的感觉。我说不清这俊秀感是由哪些艺术因素热化形成的，但我有这种感觉。同作者选择的情境有关吧？同作者选择的那些具有色彩、声响的状语有关吧？同作者对事物独特的感觉、印象、联想、想象有关吧？是的，都有关。

**婉曲**：也有两层意思，一是用意表达上的婉曲，一是诗情内涵及其流动态势的婉曲。李瑜有不少的诗，是婉曲的。诗中情思的脉动，不像人工渠水，直通通的；而像自然的溪流，有那么几个弯儿。即如只有六行的《啊，你还是不要离开我》，那情思线也不是直线。试看：“你到闪光的露珠干涸了再离开我吧，你到美丽的夜莺飞去了再离开我吧，你到黯淡的星光消逝了再离开我吧。你到朦胧的残月西坠了再离开我吧；啊，你还是不要离开我，你最好还是不要离开我！”不是就有几个弯吗？情思线一曲折，就曲折出味儿来。

李瑜在语言的运用上好像从维吾尔族民歌汲收了点什么。维吾尔族民歌，我读得甚少。读着集子里不少的诗，总觉得有些地方有维吾尔族民歌味。“汗水付与茫茫的瀚海了，心血付与茫茫的瀚海了，什么都付与茫茫的瀚海了。只留下悲怆的遗言，只留下美丽的憧憬，只留下绿色的梦幻。”这般的排句、句式、节奏，在他的诗里，不时出现，使他的诗添了民歌味，多了点新疆色彩。过去许多诗人都曾从有着永久魅力的民间创作中汲取营养。当代诗就不能从民间文学汲取营养吗？能的。这有利于诗的群众化、民族化。但，这要看诗人的意愿，来不得勉强。也勿须勉强。李瑜从维吾尔族民歌汲取了点东西，看来和他诗的总格调还是统一的，谐调的。

如果说李瑜诗歌情思的质地，那是刚性的。诗里活跃着的是开拓的理想、开拓的勇敢、开拓的韧性，而且有一些诗里也有刚性的意境、形象。但他的诗给人的总体审美感觉还是自然、俊秀、婉曲。李瑜使自己的诗具有这样的特征，在新疆众多诗人的画廊里就能够占有自己的位置。

### D

我想说：李瑜的诗已大体形成自己的艺术特征，但好像他也有另外的追求，向同自己的艺术特征并不谐调的方面靠。不一定这样吧？至少，不一定现在就这样吧？使自己的诗具有自己的艺术特征、风格，不那么容易。何不稳定、巩固一个时期呢！

我想说：有些诗内涵还欠丰富、深厚；对生活的思索还显得表面些。我们的生活并非无差别境界，诗人也不必视而不见，见而无动于心。

我想说：该让每一个词儿都十分熨贴，艺术嘛，少不得精雕细琢。比如形象的雷同就是应注意的。描绘大漠，一会儿说：“轻抚凝固了的沙包的波涛”；又一会儿说：“凝固了的沙浪又在眼前飘浮”。一而再，读者就会觉得诗人吝啬自己的智慧。

我想说：艺术手法还可以更多样些。

### E

萨克雷《名利场》有段话说：“一个人受了友谊的感动去办事的时候，本来胆小的变得勇敢了，本来怕羞的有了自信了，懒怠的也肯动了。”

是李瑜的挚情使我由怵到勇。

生孩子要节育，一对夫妻一个孩。

写诗，多多益善！李瑜的处女集诞生了，我盼着他的第二个、第三个……他正当壮年，好时候哪！

**1984年5月6日 沈阳辽宁协作**
