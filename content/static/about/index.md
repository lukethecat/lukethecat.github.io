# 诗人李瑜

## 生平简介

**李瑜**，当代著名西部诗人。

- **原籍**：安徽巢县
- **出生**：重庆
- **成长**：武汉
- **创作生涯**：一生中大部分光阴是在天山北麓的风雪和浓荫中度过

**诗歌风格**：忧伤、缠绵、轻柔、淡远，为"新边塞诗"的多样性和丰富性作出了特殊贡献。

## 代表作品

### 诗集
1. 《准噶尔诗草》
2. 《啊，伊犁河水漂白了我的军衣》
3. 《战争与城》
4. 《为了爱情，巴格达不嫌远》
5. 《汗血马》（1995）
6. 《黑罂粟》

### 散文集
- 《小巷系列》

## 创作特点

李瑜的诗歌创作生涯曾由中国国际广播电台以十多种语言对外播发介绍，并翻译成多种少数民族文字。作品以西域、中亚、丝绸之路上的历史
