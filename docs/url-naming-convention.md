# 文章 URL 命名规范

## 格式
`{作者}-{主题}-{年份}`

## 规则
1. **作者**：主要作者姓名拼音（小写）
2.**主题**：文章核心主题的简短描述（英文或拼音）
3. **年份**：发表年份

## 示例
- 孤岛写李瑜印象（1994）→ `gudao-liyu-impression-1994`
- 田先瑶悼念李瑜（2024）→ `tian-xianyao-liyu-memorial-2024`
- 张秋格新边塞诗研究（2011）→ `zhang-qiuge-xinbianseshi-2011`
- 夏冠洲论李瑜诗歌（2011）→ `xia-guanzhou-liyu-poetry-2011`

## 特殊情况
- 书序/跋：`{书名拼音}-{类型}` (如 `hanxuema-preface`)
- 研究论文：`{作者}-{研究主题}-{年份}`
- 悼念文章：`{作者}-miss-{对象}-{年份}`

## 禁止
- ❌ 不要使用错误的年份
- ❌ 不要使用模糊的来源名（如 "xinjiang-economy"）
- ❌ 不要基于编造的内容命名
