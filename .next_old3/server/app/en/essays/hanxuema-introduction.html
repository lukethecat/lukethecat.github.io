# 内容简介

《汗血马》收入李瑜西部历史诗21组101首。诗集重现了自西汉初年至鸦片战争约2000年间活跃在西域历史舞台的著名人物。那奔腾呼啸的汗血马是生命的象征和力量的化身，是血气沸沸扬扬的挥洒，是雄浑博大的民族精神的再生。诗人在特殊的艺术创造中传达了鲜明的时代共振的脉动。此艺术魅力不产生于单纯的西域风骨景观的表层，而是那具有浓郁地域色彩的意象凸现和组构中感到富有时代精神的心灵召唤。作者提议所有的诗人都跨上那强壮雄劲的"汗血马"，呼唤博大雄峻的民族之魂在古老的大陆悄悄复归。
